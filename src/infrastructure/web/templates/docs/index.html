🚀 Guía de Inicio Rápido

  1. Abre @usipipo_bot en Telegram
  2. Inicia el bot con /start
  3. Usa /menu para abrir la Mini App
  4. Crea tu primera clave VPN

📱 Configurar WireGuard

  1. Instala la app WireGuard (iOS/Android)
  2. Copia la configuración desde la Mini App
  3. Importa en WireGuard
  4. Conéctate

🌐 Configurar Outline

  1. Instala la app Outline Manager
  2. Copia el access key desde la Mini App
  3. Importa en Outline
  4. Conéctate
