# Contributing to uSipipo Landing

¡Gracias por tu interés en contribuir a uSipipo!

## Cómo Contribuir

1. **Fork** el repositorio
2. Crea tu rama de feature (`git checkout -b feature/NombreFeature`)
3. Commit tus cambios (`git commit -m 'Add: Descripción'`)
4. Push a la rama (`git push origin feature/NombreFeature`)
5. Abre un **Pull Request**

## Estándares de Código

- Follow el estilo del código existente
- Asegúrate que los tests pasen
- Actualiza la documentación si es necesario

## Preguntas?

- Abre un issue para bugs o features
- Únete a nuestro canal de Telegram: [@uSipipo_Bot](https://t.me/uSipipo_Bot)

## Código de Conducta

Por favor sé respetuoso y profesional en todas las interacciones.
